   2. 创建项目（或使用现有项目）
   3. 复制三个密钥：
      - Project URL
      - Anon Key
      - Service Role Key

**第2件：配置环境变量**
   1. 编辑 backend/.env （粘贴 Supabase 凭证）
   2. 编辑 .env （粘贴 Supabase 凭证）

**第3件：启动应用**
   打开3个终端：
   
   终端1：
   cd backend && npm install && npm run dev
   
   终端2：
   npm install && npm run dev
   
   终端3：
   curl http://localhost:3001/health

👉 第4步：打开浏览器

最常见的问题
================================================================================

Q: 后端启动失败？
A: npm install 在 backend 目录中，检查 .env 文件是否有效

Q: 前端启动失败？
A: npm install 在项目根目录，检查 .env 文件

Q: API 返回 CORS 错误？
A: 检查 backend/.env 中的 FRONTEND_URL 是否为 http://localhost:5173

Q: 无法连接到 API？
A: 确保后端在运行，检查终端1中是否有错误

Q: 看不到数据？
A: 检查 backend/data/ 文件夹是否存在（85个文件）
